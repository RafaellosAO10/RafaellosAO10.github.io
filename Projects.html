First Robot

O "First Robot" é um projeto inovador que combina a funcionalidade de um braço robótico montado sobre uma plataforma 4WD, controlado remotamente via interface web.

Detalhes Técnicos:

  • Controle via Wi-Fi: O "First Robot" utiliza a conectividade Wi-Fi da ESP32, criando um ponto de acesso próprio para permitir o controle remoto via página web. A interface web permite que o usuário manipule o braço robótico e controle o movimento do robô através de comandos simples, enviados diretamente para a ESP32.
  • Estrutura e Movimento: O braço robótico é composto por quatro servos-motores (Garra, Ombro, Cotovelo e Base), permitindo movimentos precisos em múltiplos eixos. A base motorizada é controlada por uma Ponte H L298, que gerencia os quatro motores DC responsáveis pela locomoção do robô em todas as direções (frente, trás, esquerda e direita).
  • Hardware Utilizado:
    • ESP32 32S: Microcontrolador responsável pelo processamento e conectividade.
    • Ponte H L298: Controladora de motores que permite o movimento bidirecional dos motores DC.
    • 4 Servo Motores: Responsáveis pelos movimentos do braço robótico.
    • Jumpers M/M, M/F e F/F: Conectores usados para a integração dos componentes.
    • Kit Braço Acrílico e Kit Carrinho 4WD: Estruturas físicas que formam o robô.
    • Limitador de Tensão Step Down: Protege e estabiliza a alimentação da ESP32.
    • Botão Gangorra Liga/Desliga: Fornece um controle simples para ligar e desligar o robô.

Funcionalidades:

  • Controle do Braço Robótico: Através da interface web, o usuário pode controlar cada um dos servos-motores para realizar operações complexas, como agarrar, levantar e girar objetos.
  • Movimentação: A base 4WD permite ao "First Robot" mover-se em diferentes direções, com comandos enviados através da interface web, controlando a velocidade e direção dos motores.
  • Interface Web Amigável: A página HTML desenvolvida para o controle do robô é estilizada com CSS e utiliza JavaScript para enviar comandos à ESP32, oferecendo uma experiência de controle suave e responsiva.

O "First Robot" é uma excelente demonstração de como integrar hardware e software para criar uma solução robótica funcional e acessível. Este projeto serve como uma base sólida para futuros desenvolvimentos em automação e robótica, podendo ser expandido com sensores adicionais e algoritmos de controle mais avançados.
